## Моя идея для проекта:
> Сайт с идеями и подробными статьями «Как остановить глобальное потепление»
Суть: Информационный ресурс с категориями: дома, транспорт, питание, энергия. Каждая статья содержит конкретные шаги, научные данные и простые иллюстрации.

Связь с глобальным потеплением: Самая прямая — просвещение. Чем больше людей узнают о решениях, тем выше шанс замедлить изменения климата.

Тип моего проекта:
Информационный веб-сайт с статьями, научными данными и простыми иллюстрациями. Цель — просвещение и мотивация к действию.

Библиотеки, которые я буду использовать

AOS (Animate on Scroll) — для плавных анимаций карточек 

Font Awesome — для иконок

Google Fonts — для нестандартных шрифтов

Референсы, которые мне пригодятся
Climate Action Tracker — стиль подачи данных и трекеров действий (https://climateactiontracker.org)

Project Drawdown — оформление категорий решений и их потенциала (https://drawdown.org/solutions)

Гайды-статьи, где есть полезная для меня информация
